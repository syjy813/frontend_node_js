## 1. 디렉토리

- cd: 디렉토리 이동
  - .: 현재 디렉토리
  - ..: 상위 디렉토리

```bash
cd[절대경로]
cd /path/to/file #이 경로로 이동
cd[상대경로]
cd java # 현재 디렉토리에 있는 java 디렉토리로 이동
```

상대 경로
- `[현재 디렉토리 위치]` + [상대 경로 디렉토리]
